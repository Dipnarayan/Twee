# Twee
Twee is one step to Artificially Intelligent Text to Speech Engine


<img src="banner.png" width="250px">

*Example Snippet*
```python
import Twee as t
t.about()
t.author()
t.say("do you like this")
t.csay("f~do f~you s~like s~this")
t.intelligence=1
t.say("do you like this")
t.intelligence=0
t.say("do you like this")
```
